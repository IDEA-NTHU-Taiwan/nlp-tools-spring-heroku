# Online Java NLP Tools

Simple Java Maven web project for NLP tools running on Spring MVC.

## Installation

Requires Java 7, Eclipse 4.5.1 (Mars), Apache Maven 3.3.3, a Java web server (e.g. Apache Tomcat 7)
